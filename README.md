This is a Network topology designed with Cisco's design software, Packet Tracer.
The topology has multiple Local Area Networks (LANs) connected with a Router. The topology features: PCs, a DNS server, 
an HTTP server, switches and routers; all of whom are configured appropriately to allow communication between the end devices.
The IP address ranges used are the private IP address ranges: 192.168.0.0/24 and 192.168.1.0/24.
To test the connectivity of the topology:
1. Open the 'Network Topology.pkt' file with Cisco's Packet Tracer software.
2. Open any one of the PCs in the topology, and choose desktop in the new window that is opened.
3. Open the web browser on the PC of your choosing, and type 'www.mysite.com' in the url box, and press the Enter button on your keyboard.
4. The website which is stored on the webserver in the other LAN will be displayed
5. You may otherwise use the ping command in the Command Prompt application to test the connectivity between devices
